PROXYM IN DETAILS

PROXYM (aka "PROXY Multiplexer") is a proxy server which routes each request you make to a random proxy from the pool.

How do I use PROXYM? Easy as 1-2-3!

  1. You make a request to https://google.com specifying PROXYM as a web proxy server.
  2. PROXYM picks a random exit node for you and...
  3. ... routes the request there!

PROXYM is different because:

  • It applies very strict rules to the pool - all servers are anonymous and regularly checked against multiple criteria.
  • It picks the fastest random proxy for every request
  • It doesn't require any additional software.
  • It doesn't collect and store your traffic
  • The whole pool is available for you - residential, datacenter and mobile IP addresses
  • It has no bandwidth limitations
  • It has an embedded API to filter the pool
  • Provides a complementary REST API
  • It's made and maintained by developers :)

Benefits of using PROXYM:

  • Full refund for unused period
  • Free trial period when you contact us
  • Pay by card, PayPal or Bitcoin

Typical PROXYM use cases:

  • Check a site from a different country
  • Access the site which is blocked in your country
  • Perform a load testing of a site
  • Crawl data from multiple IP addresses
  • Get around IP rate limits
